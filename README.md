# рҹҡҖ Getting Started

## Clone Repository

```bash

cd MindWell
```

---

## Backend

```bash
cd server

npm install

npm run dev
```

---

## Frontend

```bash
cd client

npm install

npm run dev
```

---

## ML Service

```bash
cd ml-service

pip install -r requirements.txt

python app.py
```
